## DOS 学习环境

DOS 环境模拟: DosBox, DosEmu
汇编开发工具: MASM(masm / link)
汇编调试工具: DEBUG

[masm, link, debug 打包下载](./masm-tools.rar)

- DEBUG 的主要命令

```
R ——查看和修改寄存器
D ——查看内存单元
E ——修改内存单元
U ——反汇编，将机器指令变为汇编指令
T /P——单步执行
G ——连续执行程序
A ——输入汇编指令
Q ——退出
```
